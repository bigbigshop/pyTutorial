BOOLEAN

什麼是 Boolean？

Boolean（布爾值）是一種只有兩種可能的數據類型：

✅ True（真）： 條件成立，表示「是」

❌ False（假）： 條件不成立，表示「否」

🎯 重點： Boolean 只有兩個值：True 和 False，就像開關的「開」和「關」！

SYNTAX

Boolean 的語法

使用比較運算符來產生 Boolean 值：

# 比較運算符
print(5 > 3) # True
print(10 == 10) # True
print(7 < 2) # False
print(8 != 8) # False

# 變數也可以存放 Boolean
is_student = True
is_passed = score >= 60

⚠️ 注意： `=` 是賦值，`==` 是比較是否相等！

PRACTICE 1

任務一：跟老師做

練習使用 Boolean 判斷考試是否通過：

# 學生的分數
score = 75

# 判斷是否及格
is_passed = score >= 60
print(f"分數：{score}")
print(f"是否及格：{is_passed}")

# 多個條件判斷
is_excellent = score >= 90
is_good = score >= 80 and score < 90
print(f"是否優秀：{is_excellent}")
print(f"是否良好：{is_good}")

💡 小技巧： `and` 表示「且」，`or` 表示「或」，`not` 表示「非」！

FLAG

什麼是 Flag（旗標）？

Flag 是一個用來記錄「是否發生某事」的變數：

🚩 像旗子一樣： 預設是「倒下」的狀態（False）

🎯 一旦條件達成： 旗子就「豎起」（變成 True）

🔍 用於檢查： 程式後面可以查看 Flag 來決定下一步

💡 生活例子： 就像學校的「遲到登記簿」—— 一開始是空的（False），有人遲到就打勾（True）。

APPLICATION

Flag 的實際應用

用 Flag 來檢查列表中是否有不及格的學生：

# 學生分數列表
scores = [85, 92, 45, 78, 60, 55]

# 初始化 Flag（預設為 False，表示「沒有不及格」）
has_failed = False

# 遍歷所有分數
for score in scores:
    if score < 60:
        has_failed = True # 找到不及格，豎起旗子

# 最後檢查 Flag
if has_failed:
    print("有同學不及格！需要補考！")
else:
    print("全部同學都及格！")
BREAK

什麼是 break？

break 是「立即停止循環」的指令：

🛑 立即退出： 遇到 break 就馬上停止整個循環

⏭️ 節省時間： 找到答案後不用繼續遍歷

🔗 常用組合： `while True + break` 是經典模式

# 找到第一個不及格的學生就停止
scores = [85, 45, 92, 78, 60]

for score in scores:
    if score < 60:
        print(f"找到不及格！分數：{score}")
        break # 立即停止循環
FLAG + BREAK

Flag 與 break 的完美結合

用 Flag 記錄「是否找到」，用 break 提早結束：

# 在列表中搜尋 100 分的學生
scores = [85, 72, 100, 90, 60]
found = False
position = 0 # 記錄位置

for score in scores:
    position = position + 1 # 位置從 1 開始
    print(f"檢查第 {position} 位同學，分數：{score}")
    if score == 100:
        found = True
        print(f"找到滿分學生！在位置 {position}！")
        break

if not found:
    print("沒有找到滿分學生")

💡 解釋： `position = position + 1` 每次循環位置加 1，就像數數「1、2、3...」

WHILE TRUE

經典模式：while True + break

這種組合讓程序等待用戶輸入，直到滿足條件才退出：

# 密碼驗證程序
password = "python123"

while True:
    guess = input("請輸入密碼：")
     
    if guess == password:
        print("登入成功！")
        break # 密碼正確，退出循環
    else:
        print("密碼錯誤，請重試！")

🎯 重點： `while True` 創造一個無限循環，內部的 `break` 是唯一退出方式！

PRACTICE 2

任務二：跟老師做

使用 Flag + break 在列表中找特定數字：

# 在數字列表中找第一個偶數
numbers = [7, 3, 8, 5, 2, 9]

found_even = False

for num in numbers:
    print(f"檢查數字：{num}")
    if num % 2 == 0: # % 是取餘數
        print(f"找到偶數：{num}！")
        found_even = True
        break

if not found_even:
    print("沒有找到偶數")

💡 小技巧： `num % 2 == 0` 表示「num 能被 2 整除」，即偶數！

SUMMARY

Boolean、Flag 與 Break 小總結

📋 Boolean 知識點

✅ 值： True 或 False

✅ 比較： `==`, `!=`, `>`, `<`, `>=`, `<=`

✅ 邏輯： `and`, `or`, `not`

📋 Flag + Break 知識點

✅ Flag： 記錄「是否發生某事」

✅ Break： 立即停止循環

✅ 結合： Flag + Break = 智能搜索

DEBUG

常見錯誤與避免方法

❌ 混淆 `=` 和 `==`： `=` 是賦值，`==` 是比較。

❌ Flag 忘記初始化： 使用前必須先給 Flag 一個初始值。

❌ break 放在錯誤位置： break 只會停止最內層的循環。

❌ 無限循環： `while True` 沒有 break 會永遠執行。

✅ 建議： Flag 命名要有意義，如 `is_found`、`has_error`。使用 `if is_found:` 而非 `if is_found == True:`。

scores = [85, 45, 100, 72, 58]
