BOOLEAN

什麼是 Boolean？

Boolean（布爾值）是一種只有兩種可能的數據類型：

✅ True（真）： 條件成立，表示「是」

❌ False（假）： 條件不成立，表示「否」

🎯 重點： Boolean 只有兩個值：True 和 False，就像開關的「開」和「關」！

SYNTAX

Boolean 的語法

使用比較運算符來產生 Boolean 值：

# 比較運算符
print(5 > 3) # True
print(10 == 10) # True
print(7 < 2) # False
print(8 != 8) # False

# 變數也可以存放 Boolean
is_student = True
is_passed = score >= 60

⚠️ 注意： `=` 是賦值，`==` 是比較是否相等！

PRACTICE 1

任務一：跟老師做

練習使用 Boolean 判斷考試是否通過：

# 學生的分數
score = 75

# 判斷是否及格
is_passed = score >= 60
print(f"分數：{score}")
print(f"是否及格：{is_passed}")

# 多個條件判斷
is_excellent = score >= 90
is_good = score >= 80 and score < 90
print(f"是否優秀：{is_excellent}")
print(f"是否良好：{is_good}")

💡 小技巧： `and` 表示「且」，`or` 表示「或」，`not` 表示「非」！

FLAG

什麼是 Flag（旗標）？

Flag 是一個用來記錄「是否發生某事」的變數：

🚩 像旗子一樣： 預設是「倒下」的狀態（False）

🎯 一旦條件達成： 旗子就「豎起」（變成 True）

🔍 用於檢查： 程式後面可以查看 Flag 來決定下一步

💡 生活例子： 就像學校的「遲到登記簿」—— 一開始是空的（False），有人遲到就打勾（True）。

APPLICATION

Flag 的實際應用

用 Flag 來檢查列表中是否有不及格的學生：

# 學生分數列表
scores = [85, 92, 45, 78, 60, 55]

# 初始化 Flag（預設為 False，表示「沒有不及格」）
has_failed = False

# 遍歷所有分數
for score in scores:
    if score < 60:
        has_failed = True # 找到不及格，豎起旗子

# 最後檢查 Flag
if has_failed:
    print("有同學不及格！需要補考！")
else:
    print("全部同學都及格！")
PRACTICE 2

任務二：跟老師做

使用 Flag 找出列表中是否有重複的數字：

# 數字列表
numbers = [5, 8, 3, 9, 2, 5, 7]

# 初始化 Flag
has_duplicate = False

# 檢查是否有重複
for i in range(len(numbers)):
    for j in range(i + 1, len(numbers)):
        if numbers[i] == numbers[j]:
            has_duplicate = True

# 輸出結果
if has_duplicate:
    print(f"發現重複數字！")
else:
    print(f"沒有重複數字")

💡 為什麼用 Flag？ 可以提前結束搜索，找到答案後就不用繼續檢查了！

SUMMARY

Boolean 與 Flag 小總結

📋 Boolean 知識點

✅ 值： True 或 False

✅ 比較： `==`, `!=`, `>`, `<`, `>=`, `<=`

✅ 邏輯： `and`, `or`, `not`

📋 Flag 知識點

✅ 用途： 記錄「是否發生某事」

✅ 初始值： 通常是 False

✅ 設定： 當條件滿足時變成 True

DEBUG

常見錯誤與避免方法

❌ 混淆 `=` 和 `==`： `=` 是賦值，`==` 是比較。

❌ Flag 忘記初始化： 使用前必須先給 Flag 一個初始值。

❌ 在循環中忘記更新 Flag： 導致 Flag 永遠是初始值。

❌ 使用 `=` 比較 Boolean： 應該直接用 `is_found == True` 或更好的是 `if is_found:`。

✅ 建議： Flag 命名要有意義，如 `is_found`、`has_error`、`is_valid`。✅ 簡化： `if is_found == True:` 可以簡化成 `if is_found:`。

SOLUTION

挑戰題解答

跟著老師一步步完成：

# 步驟 1：學生分數列表
scores = [85, 45, 100, 72, 58]

# 步驟 2：初始化 Flags
has_failed = False
has_perfect = False
all_passed = True

# 步驟 3：遍歷並檢查
for score in scores:
    if score < 60:
        has_failed = True
    if score == 100:
        has_perfect = True

# 步驟 4：檢查是否全部及格
for score in scores:
    if score < 60:
        all_passed = False

# 步驟 5：輸出結果
print(f"有人不及格：{has_failed}")
print(f"有人獲得满分：{has_perfect}")
print(f"全部及格：{all_passed}")
