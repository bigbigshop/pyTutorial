AUTO

什麼是「自動化」？

讓電腦重複執行相同的工作，無需人工干預。

🔄 省時間： 3 行代碼完成 100 次操作

🎯 零錯誤： 電腦不會累，不會出錯

💪 大規模： 輕易處理數千筆數據

WHILE LOOP

While 循環語法

只要條件成立，就重複執行程式碼：

count = 1
while count <= 5:
    print(count)
    count = count + 1

⚠️ 注意： 一定要有讓條件變為 False 的程式碼，否則會變成「無限循環」！

F-STRING

小知識：f-string 字符串格式化

在 Python 中，f-string 是一種方便的字符串格式化方法：

# 傳統方法
name = "小明"
age = 15
print("你好，" + name + "，你今年" + str(age) + "歲")

# f-string 方法（更簡潔）
print(f"你好，{name}，你今年{age}歲")

🎯 重點： 在字符串前加上 `f`，然後用 `{}` 包裹變數名，Python 會自動替換成變數的值！

PRACTICE 1

任務一：列出 1-50

跟著老師一起做，學會 while 循環的基本用法：

# 初始化變數
number = 1

# 設定循環條件
while number <= 50:
    print(number)
    # 每次加 1，避免無限循環
    number = number + 1

💡 小技巧： 可以用 `number += 1` 代替 `number = number + 1`，更簡潔！

PRACTICE 2

任務二：7的乘數表

使用 while 循環印出 7 的乘法表：

# 初始化變數
count = 1

while count <= 10:
    result = 7 * count
    print(f"7 x {count} = {result}")
    count += 1

📝 練習： 把 7 改成其他數字，例如 3 或 9，看看結果！

DEBUG

常見錯誤與避免方法

❌ 無限循環： 忘記更新計數器，導致程式永遠跑不完。

❌ 縮排錯誤： 循環內的程式碼沒有正確縮排。

❌ 變數名稱錯誤： 不小心打錯變數名，導致循環無法正常運作。

✅ 避免方法： 先寫小例子測試，確認邏輯正確後再擴展。

挑戰題：成績等級計算器

融合前三堂所學，製作一個自動化成績評級系統：

🎯 任務要求：

  1. 1. 使用 input() 詢問學生人數

  3. 3. 詢問每個學生的分數 (0-100)

    •   ● 80-89: B 級
    •   ● 70-79: C 級
    •   ● 60-69: D 級
    •   ● 0-59: F 級
  5. 5. 印出每個學生的分數和等級
SOLUTION

挑戰題解答步驟

跟著老師一步步完成：

# 步驟 1：詢問學生人數
student_count = int(input("請輸入學生人數: "))

# 步驟 2：初始化計數器
current = 1

# 步驟 3：進入循環
while current <= student_count:
    # 步驟 4：詢問分數
    score = float(input(f"請輸入第 {current} 位學生的分數: "))
    
    # 步驟 5：判定等級
    if score >= 90:
        grade = "A"
    elif score >= 80:
        grade = "B"
    elif score >= 70:
        grade = "C"
    elif score >= 60:
        grade = "D"
    else:
        grade = "F"
    
    # 步驟 6：印出結果
    print(f"第 {current} 位學生：分數 {score}，等級 {grade}")
    
    # 步驟 7：更新計數器
    current += 1
BONUS

進階挑戰：連續輸入直到 quit

製作一個可以重複使用的計算器：

# 無限循環，直到使用者輸入 quit while True:
    num = input("請輸入數字 (輸入 quit 退出): ")
    
    # 檢查是否要退出
    if num == "quit":
        print("程式結束，再見！")
        break # 跳出循環
    
    # 計算平方值
    try:
        number = float(num)
        square = number ** 2
        print(f"{number} 的平方是 {square}")
    except:
        print("請輸入有效的數字！")

💡 學習重點： `break` 語句可以隨時跳出循環。
