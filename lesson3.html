LOGIC

電腦的是非題

在程式中，邏輯判斷只有兩個答案：

🟢 True (真)：
條件成立，例如 10 > 5

🔴 False (假)：
條件不成立，例如 1 == 2

這就是所謂的「布林值 (Boolean)」。

IF STATEMENT

If 語法：程式的抉擇

這是引導程式走不同路徑的方式：

if 條件:
    如果 True 就執行這裡
else:
    如果 False 就執行這裡

🛑 重點： 冒號 `:` 之後必須縮排 (Indent)！

PRACTICE

實作 1：密碼檢查器

設定正確密碼為 "school123"：

pw = input("請輸入密碼: ")

if pw == "school123":
    print("歡迎進來，校長！")
else:
    print("密碼錯誤，請重試。")
COMPARISON

注意：= 還是 == ?

一個等號 `=`

賦值 (把資料放入盒子)
例：`x = 10`

兩個等號 `==`

檢查是否相等 (問問題)
例：`x == 10 ?`

BMI STANDARDS

挑戰題背景：BMI 標準

要讓 BMI 計算機變聰明，我們需要這些數值：

BMI 數值健康狀況
< 18.5過輕 (Underweight)
18.5 - 24.9正常 (Healthy)
> 25過重 (Overweight)
ELIF

新武器：elif (多重判斷)

當你有超過兩個以上的選擇時使用：

if bmi < 18.5:
    print("過輕")
elif bmi < 25:
    print("正常")
else:
    print("過重")
