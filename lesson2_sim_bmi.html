數學運算邏輯

$$BMI = \frac{體重 (kg)}{身高 (m)^2}$$

💻 程式碼結構挑戰

# 1. 取得體重並轉成數字 (w)
# 2. 取得身高並轉成數字 (cm)
# 3. 換算單位：公分轉公尺 (m = cm / 100)
# 4. 運算 BMI (提示：平方使用 **)
# 5. 輸出最後的 BMI 結果

Python 工具箱：

input 向使用者詢問資訊並獲取輸入
float 將輸入的文字轉換成「可計算的小數」
print 將計算結果顯示在螢幕上
/ 除法運算子
** 次方運算子 (例如 2**2 代表 2 的平方)
