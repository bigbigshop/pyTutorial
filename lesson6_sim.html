挑戰題：成績分析系統

任務：使用 Boolean、Flag 和 break 分析學生成績！

Boolean（布爾值）

True - 條件成立（真）
False - 條件不成立（假）

Flag（旗標）

🚩 Flag = 「監視器」變數，記錄是否發生某事

has_found = False # 預設：還沒找到
if 找到目標:
has_found = True # 豎起旗子

Break（中斷）

🛑 break = 「緊急停止鍵」，立即結束循環

for item in list:
if item == target:
break # 找到就停！
