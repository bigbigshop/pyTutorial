MATH BASICS

基礎數學運算

Python 就是一台超級計算機：

➕ 加法： `10 + 5`

➖ 減法： `10 - 5`

✖️ 乘法： `10 * 5`

➗ 除法： `10 / 2`

print(10 ** 2) # 次方 (10的2次方)
DATA TYPES

資料型態：數字 vs 文字

🔵 整數 (int)： 10, -5, 0

🟢 浮點數 (float)： 1.5, 3.14

🟣 字串 (str)： "100", "Hello"

CONVERSION

必須轉換型態！

🛑 警告：`input()` 拿到的資料是「字串」，不能直接做數學運算。

解決方法：使用 `int()` 或 `float()` 轉換。

num = float(input("輸入數字: "))
print(num * 2)
WARM UP

熱身練習：華氏轉攝氏

讓我們練習將數學公式變成程式碼。

公式：C = (F - 32) * 5 / 9

f = float(input("華氏溫度: "))
c = (f - 32) * 5 / 9
print("攝氏是：", c)
BMI LOGIC

單位換算：cm 轉 m

BMI 公式需要公尺 (m)。但我們通常習慣說 170 cm。

步驟 1： 輸入身高 (cm)

步驟 2： 除以 100 變成 (m)

cm = float(input("身高 (cm): "))
m = cm / 100
CODE LOGIC

BMI的公式

步驟 1： 要求用戶輸入體重 (kg)

步驟 2： 要求用戶輸入身高 (cm)

步驟 3： 將身高從 cm 換算成 m

步驟 4： 使用 BMI 公式計算 BMI

bmi = w / (m ** 2)

(** 表示次方)

CODE DEMO

完整的 BMI 計算機

w = float(input("體重 (kg): "))
cm = float(input("身高 (cm): "))

# 先換算，再計算
m = cm / 100
bmi = w / (m ** 2)

print("你的 BMI 是：", bmi)

DEBUG

常見錯誤 (Bug)

❌ TypeError: 忘記用 `float()` 轉換輸入就做運算。

❌ ZeroDivisionError: 除數不能為 0 (身高輸入了 0？)。

❌ 括號不對稱：公式長的時候最容易漏。
