ARRAY

什麼是「列表」？

列表（List）是 Python 中用來存儲多個數據的容器，就像一個可以裝很多東西的盒子：

📦 存多個值： 一個變數存放多個數據

🔢 有序排列： 每個元素有固定位置

✏️ 可修改： 隨時添加、刪除、修改元素

SYNTAX

列表的語法

使用方括號 `[]` 來創建列表，元素之間用逗號分隔：

# 創建一個空列表
empty_list = []

# 創建包含元素的列表
fruits = ["蘋果", "香蕉", "橙子", "葡萄"]
numbers = [1, 2, 3, 4, 5]
mixed = ["小明", 15, True, 3.14]

🎯 重點： 列表可以存放不同類型的數據！

PRACTICE 1

任務一：訪問列表元素

跟著老師一起做，學會如何取出列表中的元素：

# 創建水果列表
fruits = ["蘋果", "香蕉", "橙子", "葡萄"]

# 透過索引訪問元素（索引從 0 開始）
print(fruits[0]) # 印出：蘋果
print(fruits[1]) # 印出：香蕉
print(fruits[2]) # 印出：橙子
print(fruits[-1]) # 印出：葡萄（最後一個元素）

💡 小技巧： 使用負數索引可以從列表末尾開始計數！

OPERATION

列表常用操作

學習如何添加、修改和刪除列表元素：

# 創建列表
scores = [85, 92, 78, 90]

# 添加元素到末尾
scores.append(88)
print(scores) # [85, 92, 78, 90, 88]

# 修改指定位置的元素
scores[2] = 80
print(scores) # [85, 92, 80, 90, 88]

# 刪除指定元素
del scores[1]
print(scores) # [85, 80, 90, 88]

⚠️ 注意： 索引不能超出列表範圍，否則會出錯！

SUMMARY

列表小總結

FOR IN

什麼是 for...in 循環？

for...in 循環是專門用來遍歷序列（如列表）的循環方式：

🔄 自動遍歷： 逐一取出列表中的每個元素

📊 簡潔高效： 不需要手動管理計數器

🎯 專為序列： 最適合用於列表、字符串等

SYNTAX

for...in 循環語法

語法非常簡潔，直接遍歷列表中的每個元素：

# 基本語法
fruits = ["蘋果", "香蕉", "橙子"]

for fruit in fruits:
    print(fruit)

# 執行結果：
# 蘋果
# 香蕉
# 橙子

🎯 重點： `fruit` 是一個臨時變數，每次循環都代表列表中的一個元素！

PRACTICE 2

任務二：計算平均分數

跟著老師一起做，使用 for...in 循環計算平均分：

# 學生分數列表
scores = [85, 92, 78, 90, 88]

# 初始化總分變數
total = 0

# 使用 for...in 遍歷列表
for score in scores:
    total = total + score
    print(f"當前分數：{score}，累計總分：{total}")

# 計算平均分
average = total / len(scores)
print(f"平均分：{average}")

💡 小技巧： `total += score` 等同於 `total = total + score`，更簡潔！

CHALLENGE 2

挑戰題：列表與 for...in 綜合運用

融合本堂所學，完成挑戰：

🎯 任務要求：

  1. 1. 創建一個包含 5 個隨機分數的列表
  2. 2. 使用 for...in 循環遍歷列表
  3. 3. 計算最高分和最低分
  4. 4. 找出不及格（<60）的分數
  5. 5. 印出所有結果
# 步驟 1：創建分數列表
scores = [75, 45, 90, 58, 88]

# 步驟 2：初始化變數
highest = scores[0]
lowest = scores[0]
fail_scores = []

# 步驟 3：遍歷列表
for score in scores:
    if score > highest:
        highest = score
    if score < lowest:
        lowest = score
    if score < 60:
        fail_scores.append(score)

# 步驟 4：印出結果
print(f"最高分：{highest}")
print(f"最低分：{lowest}")
print(f"不及格分數：{fail_scores}")
DEBUG

常見錯誤與避免方法

❌ 索引超出範圍： 嘗試訪問不存在的索引位置。

❌ 使用等號比較列表： `list1 == list2` 是比較內容，`list1 is list2` 是比較記憶體地址。

❌ 在循環中修改列表： 遍歷列表時刪除元素可能導致跳過某些元素。

❌ 忘記縮排： for 循環內的程式碼必須正確縮排。

✅ 避免方法： 使用 `len(list)` 檢查列表長度，遍歷前先複製列表。
