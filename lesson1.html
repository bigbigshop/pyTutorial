PYTHON BASICS

讓電腦「說話」：print()

使用 `print()` 函數來輸出訊息。

print("你想顯示的內容")

🎯 任務：

  • 印出你的英文名
  • 印出計算結果：print(10 * 10)
MEMORY

電腦的記憶：變數

變數就像「標籤盒子」。我們先命名，再存入資料。

name = "Python Cat"
score = 100

這代表我們在名為 name 的盒子裡放了一段文字。

INTERACTION

讓電腦聽話：input()

`input()` 會讓程式暫停，等待你輸入內容：

name = input("你叫咩名？")
print("Hello " + name)

💡 這是你與電腦之間真正的「對話」。電腦會記住你的輸入，並在之後使用它。

ART

實作：ASCII 字符藝術

嘗試用多個 `print()` 畫出一隻貓：

print(" /\_/\ ")
print(" ( o.o ) ")
print(" > ^ < ")

提示：空格的數量會決定圖案是否對齊！

DEBUG

為什麼程式出錯了？

❌ 漏了括號：`print "Hi"` → 應為 `print("Hi")`

❌ 引號打架：`"Hello'` → 必須成對 `""` 或 `''`

❌ 全形標點：電腦看不懂 `（ ）`，必須用 `( )`
